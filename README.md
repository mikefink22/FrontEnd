# 🚀 Mastering FrontEnd - Ecosistema de Desarrollo

Este repositorio es mi **bitácora técnica y resumen teórico** sobre el desarrollo FrontEnd. Contiene apuntes, herramientas esenciales y una guía estructurada sobre las tecnologías clave del lado del cliente, desde los fundamentos hasta herramientas de compilación modernas.

🌍 **Puedes ver la página en vivo aquí:** [https://mikefink22.github.io/FrontEnd/](https://mikefink22.github.io/FrontEnd/)

---

## 📋 Contenido del Proyecto

El sitio está organizado en secciones estratégicas para facilitar el aprendizaje y la consulta rápida:

### 1. Fundamentos (HTML & CSS)

* Estructura semántica, manejo de imágenes, tablas y formularios.
* Uso de preprocesadores modernos como **SASS** y **Stylus**.

### 2. Lógica y DOM (JavaScript)

* Conceptos Core de JS.
* Estructuras avanzadas: **Set y Map**.
* Manipulación del **DOM** y lógica de programación.

### 3. Frameworks y Librerías

* **CSS:** Bootstrap, Foundation y Materialize.
* **JavaScript:** React JS, Angular y Vue JS.

### 4. Herramientas de Entorno (Build Tools)

* Configuración de empaquetadores como **Webpack**.
* Compilación de código moderno con **Babel**.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5** - Estructura semántica.
* **CSS3** - Diseño responsivo y variables nativas.
* **JavaScript (ES6+)** - Lógica dinámica.
* **GitHub Pages** - Hosting del proyecto.

---

## 📂 Estructura de Archivos Principal

* `index.html`: Punto de entrada principal con el stack tecnológico.
* `styles.css`: Estilos personalizados y diseño responsivo.
* `/Resumen teorico/`: Documentación detallada sobre HTML y CSS.
* `/JavaScript/`: Ejercicios y apuntes sobre lógica y manipulación del DOM.
