IMÁGENES EN HTML

Volver a inicio

Uno de los aspectos más vistosos y atractivos de las páginas web es el grafismo

Ventajas

  • Explicar objetivos
  • Organizar la información
  • Página más atractiva

Desventajas

  • Distracción (uso en exceso)
  • Aumentan tiempo de carga de la página

Herramientas para optimizar imágenes: Paint Shop Pro, Photoshop, Gimp

La etiqueta para insertar una imagen es <img>(image). Esta etiqueta no posee su cierre y en ella debemos especificar
el paradero de nuestro archivo gráfico mediante el atributo src(source)

Sintaxis: <img src="ruta_al_archivo.jpg">

Para expresar el camino se hace de la misma forma que en los enlaces HTML

Otros Atributos:

  • ATRIBUTO ALT

    • <img src="ruta_al_archivo.jpg" alt="Descripcion de esta imagen">. Dentro de las comillas se puede colocar una breve descripción

    Utilidades:

    • Posicionamiento en buscadores
    • Uso para aplicaciones para personas con discapacidad
    • Muestra la descripción durante el proceso de carga (o aún si no carga)
    • Es indispensable usarlo cuando la imagen se usa como cuerpo de un enlace

    IMAGENES QUE SON ENLACES Y ATRIBUTO BORDER

    Una imagen puede servir de enlace al igual que un texto

    El problema al hacer esto es que en ciertos navegadores se crea un borde en la imagen, del mismo color que el configurado para los enlaces.

    Sin embargo, en HTML podemos indicar que la imagen tenga borde mediante el atributo "border" que define el tamaño en pixeles del
    cuadro que rodea a la imagen.

    Para eliminar el borde hacemos: border="0"
