TABLAS EN HTML

Volver a inicio

Una tabla es un conjunto de celdas organizadas dentro de las cuales podemos alojar
distintos contenidos. HTML dispone de una gran variedad de etiquetas para crear tablas,
con sus atributos.

Sirven para mostrar datos de una manera sencilla de leer, dispuestos en filas y columnas.

ETIQUETAS BÁSICAS PARA TABLAS EN HTML

Las tablas son definidas por las etiquetas <table> y su cierre

Dentro de estas 2 etiquetas colocaremos todas las demás etiquetas de las tablas, hasta
llegar a las celdas. Dentro de las celdas colocaremos los textos y/o imágenes.

Filas: Definidas por la etiqueta <tr> y su cierre (table row).

Dentro de cada línea, habra direntes celdas. Cada una de estas celdas será definida por
otra etiqueta <td> (table data). Dentro de ella y su cierre es donde colocamos nuestro
contenido, el contenido de cada celda.

Ejemplo (usando también la etiqueta table header <th>; y sin bordes):

Titulo Celda1 Titulo Celda2
Celda1, línea1 Celda2, línea1
Celda1, línea2 Celda2, línea2
