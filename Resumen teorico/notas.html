Atributos Básicos de las Etiquetas

Atributo Descripción
id="texto" Establece un indicador único a cada elemento
class="texto" Establece la clase CSS que se aplica a los estilos del elemento
style="texto" Aplica de forma directo los estilos CSS de un elemento
title="texto" Establece el título del elemento (mejora la accesibilidad)

EMMET

Para crear una lista con la etiqueta strong dentro sin tener que escribir cada elemento, usando un atajo, una forma de hacerlo es usando la extensión Emmet para Visual Studio Code, que te permite escribir abreviaturas de HTML y CSS y expandirlas en código.

Por ejemplo, si escribes ul>li*3>strong y presionas Tab, se creará el siguiente código:

<ul>
<li> <strong> </strong></li >
<li> <strong> </strong></li >
<li> <strong> </strong></li >
</ul>

SECTORIZAR "body"

Una forma de sectorizar mejor el "body" es dividirlo en secciones usando la etiqueta <section> de HTML. Esta etiqueta define una sección semántica en un documento, como una introducción, un capítulo, una subsección, etc. Puedes usar CSS para darle estilo y posición a cada sección, y así crear un diseño más organizado y atractivo.

En el resultado 1 puedes ver cómo dividir el body en cinco secciones iguales usando la etiqueta <section> y el atributo width: 20%. También puedes usar otros valores de porcentaje o de unidades absolutas (como píxeles) para ajustar el tamaño de cada sección según tu preferencia.

En el resultado 2 puedes ver cómo dividir el body en dos partes usando la etiqueta <div> y el atributo height: 50%. También puedes usar la propiedad background de CSS para darle un color diferente a cada parte.

En el resultado 3 puedes ver algunos ejemplos de cómo dividir el body en tres secciones: una cabecera, un contenido y un pie de página.
Puedes usar la propiedad position de CSS para fijar la posición de cada sección y evitar que se solapen.

EXPRESIONES REGULARES

Una expresión regular es una forma de describir un patrón de caracteres que se quiere buscar o reemplazar en un texto.

Por ejemplo, en el código usa el método array.split() para dividir un array por un separador, que será una expresión regular.

En este caso, la expresión regular es /[aeiou]/, que significa “cualquier vocal”. Entonces, el método array.split() va a separar el array por cada vez que encuentre una vocal, y va a devolver un nuevo array con los fragmentos que quedan entre las vocales. Por ejemplo, si el array original es [“manzana”, “banana”, “naranja”, “pera”], el nuevo array será [“m”, “nz”, “n”, “b”, “n”, “n”, “r”, “nj”, “p”, “r”].

Puedes probar el código en la consola y como puedes ver, cada vocal ha sido eliminada y solo quedan las consonantes.
