Dados los arrays definidos anteriormente en nuestro ejemplo, podemos usar la consola para aplicar diferentes funciones sobre él como:

  • array.length: nos devuelve la longitud de nuestro array.
  • array.reverse(): invierte el orden de nuestro array.
  • array.forEach((e)=>console.log(e.nombre)): es para recorrer cada elemento de nuestro array que definimos mediante (e).
  • array.map((e)=>e*3): es una función que se aplica a cada elemento del array para modificarlo. Hay que guardar el resultado.
  • personas.filter(function(e){} ): Devuelve por consola un nuevo array con las condiciones que definimos en function(e); por ejemplo:

    var personasMayores = personas.filter(function(persona) {
    return persona.edad >= 18;
    });
